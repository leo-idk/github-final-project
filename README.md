# Simple Interest Calculator

A simple bash script that calculates simple interest given principal, annual rate of interest and time period in years.

## Description
This project is a command-line tool written in Bash to help users calculate simple interest easily. It is part of the final project for the Git and GitHub introduction course.

### Formula Used:
$$I = \frac{P \cdot R \cdot T}{100}$$

Where:
* **P** = Principal amount
* **R** = Annual rate of interest
* **T** = Time period in years

## How to Run
1. Give execution permission to the script:
   ```bash
   chmod +x simple-interest.sh

```

2. Run the script:
```bash
./simple-interest.sh

```



## Example of Usage

```text
Input:
- Principal: 1000
- Rate of interest (annual): 5
- Time period (years): 2

Output:
- Simple Interest: 100

```
